## Description
A static web page for Al Cruz Services, a home renovation company on Oahu that installs bi-fold doors and renovates bathroom & kitchen spaces.

Al Cruz Services webpage - https://alcruzservices.netlify.app/

## Features
* Static webpage deployed with Netlify
* WebP image format
